# installation instructions:
install the following packages: pyspark, nltk

# Part Two

to run this part you will need to use the following: 
```
spark-submit partTwo.py
```

The program will automaticaly download the required file to make the train and test data from. It will then train the calssifier and classify the test cases. As of right now the test split is 80% 20% but can be changed in the file to be whatever you want. The classifier will then test itself and output all measurements to the terminal.
